# SecretHSCodes
Российские власти давно научились скрывать общественно значимые данных: ЕГРЮЛ закрыл доступ к данным о владельцах «стратегически важных» предприятий, Росреестр - к имуществу коррупционеров. Кажется, впервые российское государство начинает напрямую удалять данные, существенно влияющие на адекватность государственной статистики. Как выяснил The Insider, в 2025 году российская таможня перестала отражать в централизованной базе данных значительное число категорий товаров.

В данном репозитории содержится список таможенных кодов, сведениях по которым с 2025 года перестали отображаться в таможенных данных в России. 

Полный текст расследования The Insider - ["Сокрытие следов поступления: российская таможня начала скрывать данные об импорте подсанкционных товаров"](https://theins.press/inv/287648).

-----

**`secrethscodes.csv`**

| Поле      | Описание                                                                 |
|-------------------|-----------------------------------------------------------------------------|
| `hs_code`        | Код товара в соответствии с [Гармонизированной системой описания и кодирования товаров](https://www.trade.gov/harmonized-system-hs-codes)                            |
| `desc_ru`             | Описание товара на русском языке   |
