# SecretHSCodes
Российские власти давно научились скрывать общественно значимые данных: скажем, ЕГРЮЛ закрыл доступ к данным о владельцах «стратегически важных» предприятий, а Российская государственная библиотека стала засекречивать диссертации чиновников, которые испугались, что в них выявится плагиат. Но впервые российское государство начинает напрямую цензурировать данные, существенно влияющие на адекватность государственной статистики. Как выяснил The Insider, в 2025 году российская таможня перестала отражать в централизованной базе данных значительное число категорий товаров

В данном репозитории содержится список таможенных кодов, сведениях по которым с 2025 года перестали отображаться в таможенных данных в России. 

Полный текст расследования The Insider - ["Сокрытие следов поступления: российская таможня начала скрывать данные об импорте подсанкционных товаров"](https://theins.press/inv/287648).
